Programming

What can I do?

On the side, I have been teaching myself to code in languages such as Kotlin for Android app development and HTML, CSS and JavaScript for web development. I really like the potential it gives you to be able to create something new such as new apps and programs and it gives me a way to channel my creativity into creating something unique. At university I've been learning how to use Matlab and Simulink to build simulations for engineering systems.

Some stuff I've done

Over the summer I completed a Android development and Kotlin masterclass online by Tutorials.eu by Denis Panjuta where I learned about the fundamentals of Kotlin and object-oriented programming and how to create apps. I chose Kotlin over Java because although Java is more widely used, Kotlin is being pushed by Google to be the future language for android development. I have created a number of small apps and a couple of larger projects, I don't have anything on the app store yet as I am still adding to the apps and debugging these processes do take time, however I have provided small sample videos of the apps below.

The above videos show two of my larger projects which are still in the making, Drinks & Dragons and Happy Hour. Drinks & Dragons is a fun drinking game rpg which involves 1-6 players who fight monsters and acquire treasure on their way to gaining 6 status points to become a "total legend", this game was very fun to create and play with my friends. It has been taking extra long because I have drawn all the artwork by hand myself and I am still adding more to the game little by little.

Happy Hour is a cocktail and drinks app which includes many premade recipes which can be searched by category, name and ingredient, acquired from TheCocktailDB. It also includes a create drink function and customisable user profiles. This app uses Firebase Firestore to store images and data online.
